# Atividade MongoDB - Oscar Database

Repositório contendo a resolução de uma atividade prática de MongoDB utilizando uma base de dados com informações históricas do Oscar.

## Conteúdo

- Consultas básicas e avançadas
- Operações de atualização
- Aggregation Framework
- Análises estatísticas e históricas
- Dashboard executivo

## Tecnologias

- MongoDB
- MongoDB Compass
- MongoDB Shell (mongosh)

## Objetivo

Praticar consultas, agregações e manipulação de dados em MongoDB através de análises sobre o Oscar.
